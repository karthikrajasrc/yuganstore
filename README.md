YugansProduct is a full stack e-commerce web application developed using the MERN stack. The application allows users to browse products, view product details, and manage products using a modern, responsive user interface with a scalable backend architecture.

This project was built as part of MERN Full Stack Development training and focuses on implementing real-world e-commerce functionality.

---

## Features

- User-friendly and responsive UI
- Product listing and product detail views
- Backend RESTful APIs
- CRUD operations for products
- MongoDB database integration
- API testing and data validation
- Modular and clean code structure
